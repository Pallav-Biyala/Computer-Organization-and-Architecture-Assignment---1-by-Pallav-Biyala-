The C folder includes all the source-related files for the C implementation:

1. Main Code:
   - C source file (.c)

2. Preprocessed Code:
   - C preprocessed output (.i), generated using gcc -E

3. Assembly Code:
   - Assembly language file (.s), generated using gcc -S

4. Executable (Machine Code):
   - Compiled binary executable produced after linking including all optimizations(e.g., mm_O0, mm_O2, mm_O3)
